Overview

This project is a modern landing page with a contact form connected to a Java Spring Boot backend. It demonstrates a complete full-stack workflow, handling frontend inputs via POST requests and processing them on the server side. This project was completed as part of the Java Full Stack Web Development Internship at Maincrafts Technology.

Features
Responsive Landing Page: Clean and mobile-friendly design using HTML & CSS.
Contact Form: Users can submit their Name, Email, and Message.
Backend Integration: Spring Boot handles form submissions and processes data on the server.
Controller Implementation: RESTful controller setup to manage form POST requests.
Full-Stack Workflow: Illustrates how frontend and backend interact seamlessly in real-world applications.

Tech Stack
Frontend: HTML, CSS
Backend: Java, Spring Boot
Build Tool: Maven
Development Environment: IntelliJ IDEA / VS Code (optional)

How to Run
Clone the repository:
git clone <repository-url>


Navigate to the backend folder:
cd backend/landingpage


Run the Spring Boot application:
mvn spring-boot:run


Open the frontend in a browser:
Open frontend/index.html and test the contact form. Submitted data will be processed by the Spring Boot backend.
